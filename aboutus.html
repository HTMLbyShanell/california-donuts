ABOUT US

This website is Assignment # 4 of my Introduction to Web Development with HTML & CSS course. The assignment was to accomplish four things: Develop a multi-page website template by building a similiar layout for each page of the website. Second, incorporate text using Lorem Ipsum for paragraph body. Thirdly, incorporate icons using a Font Awesome kit. Lastly, add some text shadows. I chose a Donut Theme so I could practice using dark and pastel hex color codes for styling.

Next semester, I look forward to learning about Responsive Web Design so I can be able to make my site look good across all devices.
